# DC TipTop | Full-Stack Business Management System

A professional, commercial web application designed and developed for a cleaning service business. This project demonstrates a full-stack approach, combining a robust Java backend with a clean, responsive frontend.

## üõ† Tech Stack

### Backend:
* **Java 17** with **Spring Boot 3.2.0**
* **Spring Security** (Authentication & API protection)
* **Spring Data JPA** (Database management)
* **MySQL** (Relational database hosted on Hostinger)
* **Java Mail Sender** (Integration for automated contact forms)
* **Cloudinary API** (Cloud-based image management and optimization)

### Frontend:
* **Vanilla JavaScript (ES6+)**
* **HTML5 & CSS3** (Responsive Design)
* **Thymeleaf** (Server-side templating)

---

## ‚öôÔ∏è Key Features

* **Secure Contact System:** A fully functional contact form that validates user input and sends automated emails via SMTP.
* **Media Management:** Integration with Cloudinary allows for high-performance image loading without slowing down the server.
* **Production Deployment:** Unlike many portfolio projects, this app is manually deployed on a **Linux-based VPS/Shared hosting** via SSH/FTP, showing real-world DevOps experience.
* **Clean Architecture:** Separation of concerns between the business logic (Spring Boot) and the presentation layer.
